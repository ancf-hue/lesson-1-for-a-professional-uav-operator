## 📖 О проекте

**«Урок 1: Обнаружение вражеского БПЛА»** — это первый урок профессиональной серии симуляторов для подготовки операторов беспилотных летательных аппаратов. Интерактивный 3D-симулятор позволяет юным пилотам в увлекательной форме освоить навыки обнаружения, захвата и передачи координат вражеского БПЛА на местности.

### 🎯 Цели проекта:
- ✅ Профориентация детей в сфере БПЛА
- ✅ Развитие пространственного мышления и координации
- ✅ Изучение основ тактической разведки
- ✅ Подготовка кадрового резерва для ВС РФ
- ✅ Военно-патриотическое воспитание молодёжи

## 🌟 Особенности проекта

### 🎮 Геймплей
- 🔍 **Поиск вражеского БПЛА** на обширной местности
- 🎯 **Система захвата цели** — удержание в прицеле 2 секунды
- 📡 **Передача координат** на командный пункт
- 💨 **Динамический ветер** с порывами — влияет на управление
- 🌫️ **Тепловизор** для обнаружения БПЛА в сложных условиях
- 📻 **Детектор сигналов** — определение направления на цель
- 🔭 **Оптический зум** до 4x

### 🎨 Визуальные эффекты
- ✨ **Стеклянные 3D кнопки** с переливающимся градиентом
- 🌅 **Реалистичное освещение** заката
- 🌲 **250 деревьев** и 5 домов на карте
- 🗺️ **Мини-карта** с отображением цели
- 💥 **Визуальные эффекты** столкновений и захвата
- 🎯 **Прицел** с системой захвата

### 🔊 Звуковая система
- 🎵 **Фоновая музыка** — оригинальная песня «Воронежский край»
- 🔊 **Звуковые эффекты** через Web Audio API
- 💨 **Звук ветра** с динамической громкостью
- 🎯 **Звуки захвата**, передачи, победы и поражения
- 🔇 **Возможность отключения** звука (клавиша M)

### 📱 Адаптация
- 📱 **Полная поддержка мобильных устройств**
- 🕹️ **Виртуальный джойстик** для управления
- 🎮 **Сенсорные кнопки** действий
- 👆 **Pinch-to-zoom** для масштабирования
- 📳 **Вибрация** при захвате цели (мобильные)

### ⚡ Оптимизация
- 🚀 **Мгновенная загрузка** стартового экрана
- 🎯 **3D сцена создаётся** только по клику на «НАЧАТЬ МИССИЮ»
- 📐 **Оптимизированная площадь** 2000×2000 м
- 🌲 **250 деревьев** вместо 500+
- 📏 **Ограничение высоты** 200 метров
- 🖼️ **Текстура 1024×1024** вместо 4096×4096

📋 Задание миссии


🎯 Цель:

Обнаружить замаскированный вражеский БПЛА на местности и передать его координаты на командный пункт.


⚠️ Важно:


БПЛА появляется на 15 секунд, затем исчезает на 15 секунд


Время на выполнение миссии: 5 минут


Учитывайте порывы ветра — они влияют на управление


💡 Тактика:


Используйте тепловизор для поиска БПЛА


Применяйте детектор сигналов для определения направления


Наведите прицел на цель и удерживайте F для захвата


После захвата нажмите T для передачи координат
